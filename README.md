## What This Does

- Solve Wordle faster than ever.

- This project aims to reach the **theoretical minimum average of 3.42 guesses per word** *(spoiler : it didn't ofcoarse)*.

- Includes tools to track bot‚Äôs accuracy, visualize progress, and even optional cheat helpers.

> **Note**: This is an experimental research project. Expect random changes.

---

## Features

- **Wordle Solver:** Solve any Wordle puzzle efficiently.  
- **Progress Tracking:** Interactive charts showing accuracy, average guesses, and runtime.  
- **Cheat Helper:** Optional Java-based helper for analysis.  
- **Modern GUI Launcher:** Tkinter-based launcher to run all features safely and easily.  
- **Thread-safe Execution:** Prevents crashes when running plots or helpers.
